#  Publisher MQTT - Robotat UVG

#Descripción general

Este repositorio contiene los scripts utilizados para la publicación de datos hacia un **broker MQTT**.  

El código principal —`publisher_MQTT_final.py`— se desarrolló utilizando como base las funciones y estructuras implementadas por **MSc. Miguel Zea**, adaptándolas para integrarse a la arquitectura MQTT.

---

##  Archivos principales

| Archivo | Descripción |
|----------|-------------|
| `publisher_MQTT_final.py` | Script principal desarrollado por **Sara Hernández**, que publica en tiempo real los datos del sistema de captura de movimiento  hacia el broker MQTT. Hace uso de las funciones compartidas por MSc. Miguel Zea. |
| `MoCapData.py` | Módulo proporcionado por **MSc. Miguel Zea**. **No fue modificado.** |
| `NatNetClient.py` | Proporcionado por **MSc. Miguel Zea**. **No fue modificado.** |
| `DataDescriptions.py` | Archivo proporcionado por **MSc. Miguel Zea**. **No fue modificado.** |
